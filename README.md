# Hello Grace Yoga

A React-based website for Hello Grace Yoga.

**Live site:** [https://www.hellograceyoga.com](https://www.hellograceyoga.com)

## Development

### Install dependencies
```bash
npm install
```

### Run locally
```bash
npm start
```
Opens [http://localhost:3000](http://localhost:3000) in your browser. The page reloads on changes.

### Run tests
```bash
npm test
```

## Deployment

This site is hosted on **GitHub Pages**. The `main` branch contains source code; the `gh-pages` branch contains the built site.

### Deploy to GitHub Pages
```bash
npm run deploy
```

This command:
1. Builds the production app (`npm run build`)
2. Pushes the `/build` folder to the `gh-pages` branch
3. GitHub Pages automatically serves the updated site

Changes typically appear within 1–3 minutes after deploy completes.

## Tech Stack
- React 18
- React Router
- GitHub Pages
